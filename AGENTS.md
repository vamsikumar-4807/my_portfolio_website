# Portfolio Project

This is the personal portfolio of **Vamsi Kumar Dusanapudi**, built with TanStack Start, React, TailwindCSS v4, and Vite.

## Tech Stack
- React 19 + TypeScript
- TanStack Start (SSR) + TanStack Router
- TailwindCSS v4 + shadcn/ui
- Framer Motion animations
- Vite 8

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
